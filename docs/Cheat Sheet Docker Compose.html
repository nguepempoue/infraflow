📘 Commandes Docker Compose

1. Vérifier Docker et Docker Compose

# Vérifier la version de Docker Compose
docker compose version
# (ancienne syntaxe)
docker-compose --version

2. Lancer les services

# Depuis le dossier où se trouve docker-compose.yml
docker compose up
# (ancienne syntaxe)
docker-compose up

3. Mode détaché (en arrière-plan)

docker compose up -d
# (ancienne syntaxe)
docker-compose up -d

4. Vérifier les conteneurs en cours

docker ps

5. Arrêter et nettoyer

# Arrêter et supprimer les conteneurs
docker compose down
# Supprimer aussi les volumes
docker compose down --volumes

6. Logs des services

docker compose logs
docker compose logs -f   # suivre en temps réel

7. Exécuter une commande dans un conteneur

docker compose exec  bash
# Exemple : docker compose exec web bash
